# Sistema CRUD em PHP e MySQL

Projeto acadêmico desenvolvido durante o curso de Análise e Desenvolvimento de Sistemas.

## Objetivo
Aplicar conceitos básicos de desenvolvimento web e banco de dados, implementando as operações de cadastro, consulta, atualização e exclusão de dados.

## Tecnologias utilizadas
- PHP
- MySQL
- HTML/CSS
- SQL

## Funcionalidades
- Cadastro de registros
- Consulta de dados
- Atualização de informações
- Exclusão de registros

## Status do projeto
Projeto acadêmico com foco em aprendizado e consolidação de conceitos de back-end.
